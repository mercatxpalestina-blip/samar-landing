# Samar Landing

Landing page React + Vite basada en el póster adjunto (horarios/actividades).

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

1. Sube el repo a GitHub y asegúrate de que la rama sea `main`.
2. Ve a **Settings -> Pages** y habilita GitHub Actions.
3. El workflow `/.github/workflows/deploy.yml` desplegará automáticamente el contenido desde `dist/`.
